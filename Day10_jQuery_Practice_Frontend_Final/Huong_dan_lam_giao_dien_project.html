CÁCH XÂY DỰNG GIAO DIỆN FRONTEND Vì đây là nơi hướng tới người dùng, nên cần giao diện đẹp, và responsive được sẽ là tối ưu nhất 1 - CẦn xác định chủ đề làm bài tập lớn của các bạn trước + Trang bán sản phẩm + Tin tức (ko nên làm), + Thi trắc nghiệm - up đề thi trắc nghiệm lên hệ thống, user bên frontend có thể thi + Chia sẻ file - làm 1 hệ thống cho phép nhiều user upload file lên, có cơ chế user cần trả phí mới download dc file, 2 - Đều có thể TỰ CODE hoặc Đi TÌM - đi CHÔM giao diện (search GOOGLE) r chế biến lại theo ý của mình, 1 số mẫu template có sẵn: Thương mai điện tử: https://www.free-css.com/template-categories/ecommerce Tin tức: https://www.free-css.com/template-categories 3 - Cần xác định các page mà trang của bạn có thể có Ví dụ với trang bán hàng thì sẽ có các trang: - Trang chủ - Trang chi tiêt sản phẩm - Trang giỏ hàng: liệt kê danh sách các sản phẩm đang có trong giỏ hàng - Trang checkout: Nhập thông tin thanh toán - Liệt kê danh mục sản phẩm - Trang tin tức - Đăng nhập/Đăng ký - Trang contact ... CÁCH XÂY DỰNG GIAO DIỆN BACKEND - Backend liên quan đến việc xử lý logic cho hệ thống, và sẽ chị phục vụ 1 nhóm nhỏ user , nên cũng ko quan trọng giao diện bằng frontend đc, nhưng tốt nhất vẫn cần chỉn chu - Giao diện backend thường bao gồm các form CRUD (Create - Read - Update - Delete), nên thường được tích hợp luôn Bootstrap để sử dụng các component sẵn có của Bootstrap Có thể sử dụng các template có sẵn bằng cách search, 1 template điển hình cho backend có thể kể đến là AdminLTE https://adminlte.io/..
